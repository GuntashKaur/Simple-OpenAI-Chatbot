# Simple OpenAI Chatbot (Python)

This repository contains a beginner-friendly Python project that demonstrates how to use the OpenAI API for generating text, summarizing content, and creating a custom poetic chatbot.

The project includes:
- A simple text generator  
- Customizable output settings  
- A keyword-based text summarizer  
- A poetic chatbot using conversational prompts  
- A `config.py` (not included) for securely storing  API key
